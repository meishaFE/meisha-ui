## Development

1. 请先 fork 一份到自己的项目下，不要直接在仓库下建分支。
2. 开发环境搭建
    ```shell
    git clone git@github.com:meishaFE/meisha-ui.git
    npm run dev
    ```
## npm scripts
```shell
# 开始开发
npm run dev

# 单元测试
npm run test

# 打包
npm run dist
```
